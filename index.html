My PatternLibrary

My PatternLibrary will allow elements from this to be downloaded to your own webpage!

Setup

CSS

Include this CSS file head of your HTML page to link to the relevant created stylesheet.

<link rel="stylesheet" href="css/PatternLibrary.css">

Font Awesome Icons

If you want to use icons with PatternLibrary, make sure you include the Font Awesome CSS file into the <head> of your HTML page. Icons have been added for the social media aspects in the footer element of this Pattern Library.

<link rel="stylesheet" href="https://use.fontawesome.com/releases/v5.0.12/css/all.css">

Structure of the HTML

This template includes all of the relevant files needed to get started with PatternLibrary.

<!DOCTYPE html>
<html lang="en">
    <head>
        <!-- Required meta tags for the HTML -->
        <meta charset="utf-8">
        <meta name="viewport" content="width=device-width, initial-scale=1">
        
        <!-- Font Awesome Icons -->
        <link rel="stylesheet" href="https://use.fontawesome.com/releases/v5.0.12/css/all.css">
        
        <!-- Import PatternLibrary.css -->
        <link rel="stylesheet" href="css/PatternLibrary.css">
        
        <title>Welcome to the PatternLibrary</title>
    </head>
    <body>
        
        <!-- Import jQuery first -->
        <script type="text/javascript" src="https://code.jquery.com/jquery-3.3.1.min.js" 
        <!-- Import PatternLibrary.js at end of body of the HTML this will aid in the loading -->
        <script type="text/javascript" src="js/PatternLibrary.js"></script>
    </body>
</html>
