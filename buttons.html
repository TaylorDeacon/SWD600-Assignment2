Buttons

This element will be shown in various styles and settings; this will be appliable to multiple purposes.

Introduction

Default Style

The button is an essential design element designed to look like an interactive element.

<button class="btn">Button</button>

Basic Usage

The .button class can be used on four different elements:

  • <a> Anchor tags - these are hyperlinks allowing access throughout the webpage - leading the user to not be led away from the page.
  • <button> Button tags - clickable link to allow the user to be able to naviagte throught the webpage or link to another.
  • <input type="submit"> Submit inputs - to allow the user to submit information; i.e a form.
  • <input type="reset"> Reset inputs - this could have the possibility to allow a form with information already to be quicked deleted by pressing this button.
Anchor 
<a class="btn">Anchor</a>
<button class="btn">Button</button>
<input type="submit" class="btn" value="Submit">
<input type="reset" class="btn" value="Reset">

Raised - to add a shadowed effect to your button, furthering the aesthetic value.

To make any button raised, just add the .raised class to any button along with the colour and style you want.
